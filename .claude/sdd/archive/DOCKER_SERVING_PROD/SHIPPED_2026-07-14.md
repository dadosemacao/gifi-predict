## Lessons Learned

### Process

- Cortar MVP em **packaging local** (sem Azure) desbloqueou entrega no mesmo dia.
- Judge advisory repetiu pedidos de go-live security — manter escopo explícito no DESIGN evita thrash.

### Technical

- `ServingSettings.from_yaml(**kwargs)` **vence env** → overlay `serving.docker.yaml` na imagem é necessário.
- Typer com um único comando vira `serve` (não `serve run`) — validar CLI na imagem cedo.
- Publish host `127.0.0.1` + `HOST_PORT` evita conflito com outros stacks Docker.
- `demo_mode` em `/api/release-status` vem do relatório L4, não de `demo_default`.

### Communication

- Samples b+c (API + artefatos locais) bastaram para smoke; falta de `release_ok=true` precisa ficar explícita no runbook.

### Tools

- Docker multi-stage Node→Python validou bake real (~imagem com boosting + SPA).
- Compose healthcheck com `urllib` evita depender de `curl` na slim image.
