## Lessons Learned

### Process

- Judge advisory no DESIGN (lock, NaN, joblib) evitou bugs em produção do pointer.
- Padronizar venv **após** build evita drift Python 3.14 vs 3.12 — fazer setup no ship seguinte.

### Technical

- Holdout histórico tem `Extrativo_AT`/`Carga_Alcalina` reais — avaliação cascade Modo A precisa `strict_scenario=False`.
- Champion heterogêneo por elo exige `feature_cols` no manifesto para inferência consistente.
- `release_ok=false` não deve atualizar `current_candidate.json` — separar candidato empacotado de campeão produtivo.

### Communication

- MAE ~95 no Excel real é resultado útil para Camada 4 (gap analysis), não falha de build.
- MLflow rejeitado no MVP; ADR-003 documenta evolução por marco.

### Tools

- scikit-learn + joblib + typer: adequados para Marco 1 local.
- `ruff` + `uv.lock` adicionados em v0.2.1 para reprodutibilidade.
